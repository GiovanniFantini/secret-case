## Come Iniziare

### Per i Giocatori

1. **Leggete prima le istruzioni**: Iniziate con `istruzioni-giocatori.md`
2. **Consultate il manuale utente**: `manuale-utente.md` contiene il percorso di lettura consigliato
3. **Seguite i cluster tematici**: Non leggete documenti a caso, seguite l'ordine suggerito
4. **Prendete appunti**: Scrivete nomi, date, contraddizioni, sospetti
5. **Usate i suggerimenti solo se necessario**: Provate prima a risolvere da soli

### Per i Game Master

1. **Leggete `documento-master.md`**: Contiene la soluzione completa e note di regia
2. **Preparate i materiali**: Stampate i documenti o organize la cartella digitale
3. **Configurate la verifica**: Google Form o buste fisiche con le risposte
4. **Leggete `manuale-utente.md`**: Contiene timeline, checkpoint e consigli per facilitare
5. **(Opzionale) Registrate la narrazione**: L'audio della narrazione finale è più immersivo
