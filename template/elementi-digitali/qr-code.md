## Implementazione Tecnica

### Generazione dei QR Code
- Usare un generatore di QR code gratuito
- Dimensione consigliata: almeno 2x2 cm per la stampa
- Testare la scansionabilita prima della stampa finale

### Opzioni per il Contenuto Digitale
1. **Pagina web statica**: contenuto HTML ospitato su servizio gratuito
2. **Google Doc/Drive**: documento condiviso con link diretto
3. **File PDF**: documento scaricabile
4. **Nota online**: servizio di note con link pubblico

### Suggerimenti
- Ogni QR code deve funzionare indipendentemente (no sequenze obbligate)
- Prevedere un'alternativa testuale per chi non riesce a scansionare
- Non inserire indizi essenziali SOLO nei QR code: devono essere bonus o conferme
