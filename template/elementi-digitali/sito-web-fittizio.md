# Sito Web Fittizio per Verifiche

**(Elemento opzionale)**

---

## Descrizione

Un sito web fittizio che i giocatori possono visitare per ottenere informazioni aggiuntive sul caso. Puo simulare:
- Il sito web di un'azienda coinvolta nel caso
- Un profilo pubblico di uno dei personaggi
- Un articolo di blog o forum
- Un sito istituzionale

---

- **Titolo**: [Titolo della pagina]
- **Contenuto**: [Descrizione di cio che i giocatori vedono sulla homepage]

### Pagina "Chi Siamo" / "About"
- **Contenuto**: [Informazioni sull'organizzazione o persona.

  nascosto tra informazioni apparentemente irrilevanti]

### Pagina con indizio nascosto
- **Titolo**: [Titolo della pagina]
- **Contenuto**: [Testo che contiene un indizio chiave.
  Puo essere un nome, una data, un collegamento,
  un'immagine con un dettaglio rivelatore]

### Pagina di contatto
- **Contenuto**: [Informazioni di contatto che possono
  rivelare connessioni tra personaggi]

---

## Implementazione

Per creare il sito si possono usare:
- **Opzione semplice**: Un documento PDF formattato come pagina web
- **Opzione intermedia**: Una pagina HTML statica ospitata gratuitamente
- **Opzione avanzata**: Un mini-sito interattivo con elementi cliccabili
