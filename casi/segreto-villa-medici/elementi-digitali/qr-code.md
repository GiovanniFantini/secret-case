## ISTRUZIONI PER LA CREAZIONE DEI QR CODE

### Strumenti Gratuiti Consigliati

| Strumento | URL | Note |
|-----------|-----|------|
| QR Code Generator | https://www.qr-code-generator.com | Semplice, gratuito, personalizzabile |
| QRCode Monkey | https://www.qrcode-monkey.com | Permette di aggiungere logo e colori |
| Canva | https://www.canva.com/qr-code-generator/ | Integrato nel design grafico |
| Google Chrome | `chrome://flags/#sharing-qr-code-generator` | Generatore integrato nel browser |
| Python (qrcode) | `pip install qrcode` | Per generazione automatizzata |

### Procedura di Creazione

1. **Preparare i contenuti digitali:**
   - Caricare i file audio/video/pagine su una piattaforma accessibile (Google Drive, YouTube, GitHub Pages, Netlify)
   - Annotare gli URL definitivi di ogni contenuto
   - Verificare che i link funzionino su smartphone (iOS e Android)

2. **Generare i QR code:**
   - Accedere a uno degli strumenti sopra indicati
   - Inserire l'URL del contenuto
   - Personalizzare (opzionale): colore scuro rosso bordeaux (#8B0000) su sfondo crema (#F5F0E1) per mantenere l'atmosfera del caso
   - Scaricare in formato PNG ad alta risoluzione (minimo 300x300 pixel, consigliato 500x500)

3. **Preparare la stampa:**
   - Inserire ogni QR code nell'etichetta corrispondente (vedi sopra)
   - Stampare su carta di buona qualita (almeno 120 g/mq)
   - Dimensione consigliata del QR code stampato: 4x4 cm minimo (per garantire la leggibilita)
   - Lasciare un margine bianco (quiet zone) di almeno 5 mm attorno al QR code

4. **Testare:**
   - Scansionare ogni QR code stampato con almeno 2 smartphone diversi
   - Verificare che il contenuto si carichi correttamente
   - Controllare la leggibilita in condizioni di scarsa illuminazione (coerente con l'atmosfera del gioco)

### Suggerimenti per l'Atmosfera

- Stampare i QR code su carta invecchiata artificialmente (immergere in te o caffe e asciugare)
- Aggiungere un timbro fittizio "PROCURA DELLA REPUBBLICA — REPERTO ACQUISITO"
- Inserire ogni QR code in una busta di plastica trasparente per prove (acquistabili online) con un'etichetta numerata
- Per il QR-03 (telecamere), allegare un paio di screenshot simulati in bianco e nero stampati su carta fotografica

## NOTE TECNICHE

- I QR code statici (con URL fisso) sono gratuiti e non scadono
- Se si usano servizi di abbreviazione URL (bit.ly, tinyurl), i link potrebbero scadere nel tempo
- Per eventi ripetuti, si consiglia di usare QR code dinamici (modificabili) tramite servizi premium
- La dimensione minima per la stampa e 2x2 cm, ma 4x4 cm garantisce una scansione affidabile
- Assicurarsi che la connessione internet sia disponibile nel luogo di gioco (oppure preparare versioni offline dei contenuti)
