## Istruzioni per la Stampa dei QR Code

### Generazione dei Codici

Utilizzare un generatore di QR code gratuito (es. qr-code-generator.com) per creare i codici. Se i contenuti non sono effettivamente ospitati online, è possibile:

1. **Creare pagine web fittizie** su un servizio gratuito (Google Sites, Notion, ecc.)
2. **Linkare a file PDF condivisi** su Google Drive o simili
3. **Stampare QR code non funzionanti** come elementi puramente scenografici — in questo caso, il Game Master fornirà verbalmente le informazioni ai giocatori che "scansionano" il codice

### Formattazione

- Dimensione minima consigliata: **2 x 2 cm**
- Stampare su carta bianca con bordo di almeno 3 mm
- Aggiungere sotto ogni QR code una **didascalia** descrittiva (es. *"Scansiona per visitare il sito della Fondazione Luce"*)
