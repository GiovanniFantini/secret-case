# CASO SECRET CASE: LA TRAMA

**Difficoltà:** ⭐⭐⭐⭐⭐ (Livello 5 - Estremo)
**Tema:** Noir Investigativo, Corporate Drama, Frode Ambientale
**Durata stimata:** 4-6 Ore
**Totale Documenti:** ~75 Files Interconnessi

## Introduzione per il Server
"La Trama" è il caso più complesso del framework *Secret Case*. Ambientato nella Brianza industriale (Merate), fonde tre livelli criminosi separati che generano rumore di fondo (red herrings) per disorientare il giocatore:
1. Una frode ecologica e uno sfruttamento lavorativo.
2. Un'operazione di fusione e speculazione finanziaria ad alta tensione.
3. Un omicidio d'impeto protettivo scaturito per futili motivi d'età e disperazione silente.

Il giocatore dovrà analizzare prove digitali, tabulati di cella, contratti societari, autopsie ed email, per dedurre, esclusivamente per sottrazione ed esclusione tecnologica (attraverso un incrocio di log Wi-Fi con la portata limitata del segnale router), l'identità dell'assassino invisibile.
